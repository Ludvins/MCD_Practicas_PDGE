Algoritmo de Bernstein-Vazirani¶

Estudiamos el enfoque cuntico del algoritmo de Bernstein-Vazirani.

Planteamiento del problema¶

Consideramos un natural $n \in \mathbb{N}$ fijo y conocido, una cadena binaria $s \in \{0,1\}^n$ desconocida, y la aplicacin (conocida)

$$ \begin{aligned} f:\{0,1\}^n &\to \{0,1\}\\ x &\mapsto x\cdot s, \end{aligned} $$

donde $\cdot$ es el producto escalar bit a bit mdulo 2. Es decir, si $s = s_0s_1,\dots,s_{n-1}$ y $x = x_0,x_1,\dots,x_{n-1}$, entonces

$$ x\cdot s = \sum_{i=0}^{n-1} x_is_i\, \bmod 2. $$

Resolver el problema consiste en encontrar el valor de la cadena secreta $s$.

Resolución clásica¶

El algoritmo clsico para resolver el problema se basa en la siguiente idea: dado un elemento $x^{(j)} \in \{0,1\}^n$ formado por todo $0$s excepto un $1$ en la posicin $j$-sima, se tiene que

$$ f(x^{(j)}) = s_j. $$

Entonces, si aplicamos la funcin $f$ a cada elemento del conjunto $\left\{ x^{(j)} : j=0,1,\dots,n-1\right\}$, obtendremos los $n$ bits que componen a $s$. Notamos que el nmero de consultas necesarias a $f$ es justamente $n$, el tamao de $s$. Por tanto, se trata de un algoritmo que escala de manera lineal con el tamao del secreto.

Resolución cuántica¶

Explicamos ahora el funcionamiento del algoritmo de Bernstein-Vazirani para resolver el problema utilizando computacin cuntica. La novedad de este algoritmo es que requiere una nica consulta a la funcin $f$, mejorando la eficiencia lineal del algoritmo clsico.

Utilizaremos la siguiente propiedad del la puerta de Hadamard:

Lema (Nielsen and Chuang (2010), eq. 1.50). Sea $\lvert a \rangle$ un qubit bsico de dimensin $n$ y $H^{\otimes n}$ la puerta de Hadamard de dimensin $n$. Se verifica:

$$ H^{\otimes n} \lvert a \rangle = \frac{1}{\sqrt{2^n}} \sum_{x\in\{0,1\}^n} (-1)^{ax} \lvert x \rangle. $$

Demostracin. Procedemos mediante un un razonamiento inductivo. Para $n = 1$, se comprueba fcilmente que $$ H \lvert a \rangle = \frac{1}{\sqrt{2}}(\lvert 0 \rangle + (-1)^a \lvert 1 \rangle) = \sum_{x=0}^1 (-1)^{ax}\lvert x \rangle. $$

Para $n > 1$, se tiene que: $$ H^{\otimes n} \lvert a \rangle = \bigotimes_{i=1}^n \frac{1}{\sqrt{2^n}}\sum_{x_i = 0}^1 (-1)^{a_i x_i} \lvert x_i \rangle =\frac{1}{\sqrt{2^n}}\sum_{x \in \{0,1\}^n} \bigotimes_{i=1}^n (-1)^{a_i x_i} \lvert x \rangle =\frac{1}{\sqrt{2^n}}\sum_{x \in \{0,1\}^n} (-1)^{\sum_{i=1}^n a_i x_i} \lvert x \rangle = \frac{1}{\sqrt{2^n}}\sum_{x \in \{0,1\}^n} (-1)^{a x} \lvert x \rangle . $$

Corolario. En particular se tiene que: $$ H^{\otimes n} \lvert 0 \rangle ^{\otimes n} = \frac{1}{\sqrt{2^n}} \sum_{x\in\{0,1\}^n} \lvert x \rangle. $$

Definimos entonces una puerta cuntica a partir de la funcin que buscamos:

$$ U_f\lvert x \rangle = (-1)^{f(x)}\lvert x \rangle = (-1)^{sx} \lvert x \rangle. $$

Se comprueba inmediatamente que se trata de una autntica puerta cuntica (es reversible). Ahora, utilizando que la puerta de Hadamard es su propia inversa, tenemos:

$$ \lvert 00\dots 0 \rangle \xrightarrow{H^{\otimes n}} \frac{1}{\sqrt{2^n}} \sum_{x\in\{0,1\}^n} \lvert x \rangle \xrightarrow{U_f} \frac{1}{\sqrt{2^n}} \sum_{x\in\{0,1\}^n} (-1)^{sx} \lvert x \rangle \xrightarrow{H^{\otimes n}} \lvert s \rangle. $$

Circuito Cuántico¶

La construccin del circuito cuntico se basa en tres etapas:

  1. Utilizar un qubit ms de la longitud de $s$ y aplicar una puerta $H$ a cada qubit. Aplicamos adems una puerta $Z$ al ltimo qubit, que a partir de entonces acta como qubit de signo, poniendo con signo "$-$" aquellos estados que tuvieran un $1$ como qubit ms significativo.
  2. Aplicamos la funcin $U_f$ a los qubits, lo cual consiste en alterar el bit de signo de cada estado tanto como indique la salida de la funcin.
  3. Aplicamos las puertas $H$ nuevamente para obtener $s$.

La idea es usar el entrelazamiento cuntico que implica la aplicacin de las puertas $H$, de forma que actuando nicamente sobre el ltimo qubit de signo, los cambios se "propagan" al resto. Realmente, estamos aprovechando el fenmeno de phase kickback para transmitir informacin del qubit de signo a los anteriores (ver cuaderno sobre phase kickback).

Para implementar esta idea nos basamos en las puertas CNOT, que pueden controlar a travs del qubit de control si hay un $1$ en la posicin que nos interesa, y en ese caso hacer un flip del qubit de signo.

# Construimos un circuito con n+1 qubits
q = QuantumRegister(n + 1, 'q')
c = ClassicalRegister(n, 'c')
circuit = QuantumCircuit(n + 1, n)

# Aplicamos H a todos los qubits
circuit.h(q)
circuit.z(q[n])
circuit.barrier()

# Aadimos el orculo U_f
circuit.append(Uf, range(n+1))
circuit.barrier()

# Aplicamos H de nuevo
circuit.h(q[0:n])
circuit.barrier()

# Aadimos mediciones a todo excepto al bit de signo
circuit.measure(q[0:n], c[0:n])

circuit.draw('mpl')

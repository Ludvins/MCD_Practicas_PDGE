Semisumador clásico¶

El circuito clsico para suma de dos bits (tambin conocido como semisumador) es el siguiente:

En resumidas cuentas, queremos implementar el comportamiento de la siguiente tabla:

Bit 1 Bit 2 Carry Suma
0 0 0 0
0 1 0 1
1 0 0 1
1 1 1 0

En general, para representar la suma de dos bits necesitamos un nmero de dos bits: la suma puede ser $00$, $01$, $10$. El bit ms significativo se conoce como bit de carry, que solo es $1$ cuando sumemos $1+1$; y el bit menos significativo es el bit de suma, que vale $1$ si y solo si sumamos dos bits distintos ($1+0$ $0+1$). Hecho este anlisis, vemos que la puerta XOR representa perfectamente el bit de suma, mientras que el bit de carry puede conseguirse con la puerta AND.

Semisumador cuántico¶

Queremos reproducir ahora el comportamiento del semisumador clsico con un circuito cuntico. Para ello debemos buscar puertas cunticas equivalentes a la XOR y la AND, que adems sean reversibles.

En primer lugar, planteamos un circuito con 4 qubits, donde los dos primeros representarn la entrada, y los dos ltimos la salida (concretamente, $q_2$ ser el bit de suma y $q_3$ el de carry). Para probar nuestro circuito usaremos el caso ms "difcil", que es el de sumar $1+1$, por lo que transformamos las dos entradas al estado $|1\rangle$ con una puerta $X$.

In [2]:
circuit = QuantumCircuit(4, 2)

# Input
circuit.x(0)
circuit.x(1)
circuit.barrier()

circuit.draw('mpl')
Out[2]:

Para implementar el (qu)bit de suma, que guardaremos en $q_2$, necesitamos simular el comportamiento de la puerta XOR. Para esto usaremos la puerta CNOT, de forma que podamos analizar cundo alguna de las entradas vale $1$ y la otra $0$. Como no queremos sobreescribir la entrada, podemos usar dos puertas CNOT para guardar el resultado en $q_2$:

In [3]:
# Compute 'S' bit
circuit.cnot(0, 2)
circuit.cnot(1, 2)
circuit.draw('mpl')
Out[3]:

De esta forma, en $q_2$ habr un $|0\rangle$ si $q_0=q_1$, ya que hacemos flip de $q_2$ dos veces, y en otro caso habr un $|1\rangle$ (un nico flip).

Para calcular el bit de acarreo, la puerta que usaremos es la puerta de Toffoli. Esta puerta puede describirse como una puerta CNOT con dos entradas de control, que acta sobre su tercera entrada solo cuando los dos valores de control valgan $1$. Esto es justo lo que queremos para guardar el acarreo en $q_3$.

In [4]:
# Compute 'C' bit
circuit.toffoli(0, 1, 3)
circuit.barrier()
circuit.draw('mpl')
Out[4]:

Al utilizar solamente puertas NOT y CNOT, e inicializar los qubits a $|0\rangle$ o $|1\rangle$, el circuito no presenta aleatoriedad, permitiendo sumar los valores de los bits de entrada. Podemos comprobar esto analizando el estado de los dos ltimos qubits, y viendo cmo su estado conjunto colapsa al valor deseado en este caso.

def add_bits(b1, b2):
    circuit = QuantumCircuit(4, 2)
    if b1:
        circuit.x(0)
    if b2:
        circuit.x(1)
    
    circuit.cnot(0, 2)
    circuit.cnot(1, 2)
    circuit.toffoli(0, 1, 3)
    circuit.measure([2, 3], [0, 1])
    
    return circuit

# Create circuits
c1 = add_bits(0, 0)
c2 = add_bits(0, 1)
c3 = add_bits(1, 0)
